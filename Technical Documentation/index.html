Setting Up XAMPP and MySQL on Ubuntu for Beginners
Introduction Installing XAMPP: Starting and Stopping XAMPP: Using MySQL on Ubuntu: Conclusion:
Introduction

If you’re diving into web development or database management on your Ubuntu machine, setting up XAMPP and MySQL is a great starting point. This guide will walk you through the installation process and how to use MySQL for your projects.

Installing XAMPP:

Start by downloading XAMPP from the official website click here. Choose the Linux version and follow these steps:

  • Open a terminal with Ctrl + Alt + T.
  • Navigate to the Downloads directory:
    • cd ~/Downloads

    Make the downloaded file executable:

    chmod +x xampp-linux-x64-<version>.run

    Replace <version> with the downloaded version number.

    Run the installer:

    ./xampp-linux-x64-<version>.run
    Starting and Stopping XAMPP:

    Once installed, use the following commands to manage XAMPP:

  • Start XAMPP:
    • sudo /opt/lampp/lampp start

    This launches the Apache web server and MySQL database server.

    Access XAMPP Dashboard:

    "Open your web browser and go to local host to see the XAMPP dashboard.

    For database management, visit phpmyadmin.

  • Stop XAMPP:
    • sudo /opt/lampp/lampp stop
    Using MySQL on Ubuntu:
  • Start XAMPP:
    • sudo /opt/lampp/lampp start
  • Access MySQL:
    • sudo /opt/lampp/bin/mysql -u root -p

    Press Enter if you haven’t set a password.

    Use MySQL:

    Create a Database:

    CREATE DATABASE your_database_name;
    Conclusion:

    Congratulations! You’ve successfully set up XAMPP and MySQL on your Ubuntu machine.
